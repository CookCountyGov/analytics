About this Site

This data provides a window into how people are interacting with Cook County government online. The data comes from Google Analytics accounts that track traffic to Cook County sites.

Not every County government website is represented in this data. Currently, for this Alpha release, we are including the main County web site at cookcountyil.gov, as well as our Open Data Catalog site at data.cookcountyil.gov. We will continue to pursue and add more County sites to this program over time. If you would like your County agency site data included, contact our Open Data program.

This dashboard uses code developed by the Digital Analytics Program, a US government team inside the General Services Administration, an independent federal agency. It was built largely by 18F, another GSA team. This open source project is in the public domain, which means that this website and its data are free for you to use without restriction. You can find the code for this website and the code behind the data collection on GitHub.

We plan to expand the data made available here. If you have any suggestions, or spot any issues or bugs, please open an issue on GitHub or contact our Open Data program.

* Realtime "right now" data is updated every three minutes.
